# Skill Assets

This directory is reserved for product-generated repair skills.

It is intentionally separate from the repo-level `skills/` directory:
- `skills/` contains Codex/operator skills used by developers working on this repository
- `skill_assets/` contains runtime repair knowledge generated by the harness after validated failure-repair cases

Planned layout:

```text
skill_assets/
  manifests/   # indexed metadata for skill lookup
  skills/      # current canonical skill payloads
  revisions/   # immutable historical revisions
  indexes/     # lookup and matching artifacts
```

Rules:
- only validated repairs should create or update assets here
- every skill update should create a new revision
- anything used for runtime matching should remain attributable to the originating failure case
